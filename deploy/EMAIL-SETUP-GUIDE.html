Visão Geral

Quando um candidato submete o formulário de inscrição no site, os dados são enviados para o Systeme.io CRM. O sistema cria um contacto com todos os campos e atribui a tag retiro-living-prayer. Esta tag dispara uma automação que envia um email de confirmação no idioma correto.

Limitação do plano free
O plano free do Systeme.io não permite criar tags adicionais. A automação usa o campo Idioma como condicional para decidir qual email enviar — sem precisar de upgrade.

Configuração da Automação no Systeme.io

1

Criar Automação

Ir a Systeme.io → Automations → Criar nova automação

2

Definir Trigger

Tag adicionada → tag retiro-living-prayer

3

Adicionar Condição

Contact field → Idioma → contém → "Português"

Se SIM → enviar email PT
Se NÃO → enviar email EN

4

Adicionar Notificação ao Pedro

Adicionar também uma notificação por email para pedro_santos_morais@hotmail.com com todos os dados do contacto.
