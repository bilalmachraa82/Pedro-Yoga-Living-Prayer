O que foi feito hoje — em detalhe

1. Infraestrutura do site

  • Domínio livingprayer.pt — DNS configurado no Cloudflare com CNAME para Vercel
  • SSL — Certificado HTTPS ativo automaticamente
  • Site bilíngue — Alternância PT/EN via atributo data-lang, sem páginas separadas
  • Formulário de candidatura — Validação completa, envio assíncrono, feedback visual
  • Design — Cormorant Garamond + Inter, paleta beige/verde/castanho, tom contemplativo

2. Integração com Systeme.io (CRM)

  • API subscribe.js — Função serverless na Vercel que envia todos os campos para o Systeme.io
  • Contactos criados — Com nome, apelido, email, telefone, alojamento, motivação, como soube, idioma
  • Tag automática — Tag "retiro-living-prayer" adicionada a cada novo contacto
  • Gestão de duplicados — Se o email já existe, faz PATCH (atualiza dados) em vez de criar duplicado

3. Workflow de automação de emails

O workflow "Living Prayer - Candidatura Retiro" no Systeme.io executa automaticamente:

PassoAção
1. GatilhoTag "retiro-living-prayer" adicionada ao contacto
2. DecisãoVerifica se o campo "Idioma" = "Português"
3a. Se SimEnvia email PT: "Recebemos a tua candidatura ao Living Prayer"
3b. Se NãoEnvia email EN: "We received your application to Living Prayer"
4. NotificaçãoEmail para pedro_santos_morais@hotmail.com com todos os dados do candidato
Os emails são enviados automaticamente dentro de segundos após o preenchimento do formulário. O candidato recebe confirmação na sua língua, e o Pedro recebe os dados completos na sua caixa de entrada.

4. Testes realizados

  • Submissão PT com sucesso — idioma="Português", tag atribuída
  • Submissão EN com sucesso — idioma="English", tag atribuída
  • Todos os campos personalizados preenchidos corretamente
  • Workflow ativado e processando contactos

Workflow do processo de candidatura

Como funciona de ponta a ponta quando alguém se candidata:

1. Visitante acede a livingprayer.pt → Escolhe PT ou EN → Preenche formulário → Clica "Candidatar-me"
2. Vercel serverless function envia dados ao Systeme.io via API
3. Systeme.io cria contacto com 7 campos + adiciona tag "retiro-living-prayer"
4. Workflow dispara automaticamente:
    • Verifica campo "idioma" → "Português" ou "English"
    • Envia email de confirmação na língua correta
    • Envia notificação para pedro_santos_morais@hotmail.com
5. Pedro lê a notificação e responde ao candidato em 48h
